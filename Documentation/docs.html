Introduction

  • Treatise is a collection of components which are built in pure HTML and CSS. You can copy the HTML code from code snippet and use it in your project to use that component in your layouts.

  • In each component section, code snippet has the code of the actual example which is shown above the code snippet. So be aware when you using these components you have to modify the template as per your requirements.
